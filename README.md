# SYTCS-DoS
A Denial of Sevice tool developed by the SYTCS team

## Installation

clone the repository to your desktop.

```bash
git clone github.com/sytcsofficial/SYTCS-DoS
```
Install the requirements.

```bash
pip install -r requirements
```

Run the tool

```bash
python3 main.py
```
